FAQ

Common questions about Technical Compliance.

Which frameworks are mapped?

DORA, MiCA, PSD3, AMLR, Basel IV, Consumer Duty, NIS2, FinCEN BSA, MAS DTSP, and 30+ other global frameworks — maintained by Vixio's analyst team.

Can one control satisfy multiple obligations?

Yes. One-to-many mapping is native — so a single well-designed control can evidence compliance across overlapping frameworks (e.g. DORA + NIS2).

Are control tests automated?

Tests run on a configurable cadence. Failures open workflows automatically with evidence attached and a reviewer assigned.

Can we export for regulators and auditors?

Yes. Every test has attached evidence, reviewer, and timestamp — exportable as a clean audit pack in one click.

Which sectors use this?

Payments, retail banking, digital assets, crypto, and gambling operators with complex technical-standards obligations.
